# Kasku Android WebView

## Cara Build
1. Buka folder ini di **Android Studio**.
2. Masukkan file `google-services.json` (dari Firebase Console lu) ke dalam folder `app/`.
3. Ganti URL di `MainActivity.java` ke domain Vercel lu yang udah online.
4. Klik Build -> Build Bundle / APK.

APK siap dipakai. Push notification akan menyala berkat Firebase Messaging Service.
